# JSMKC E2Eテストケース

## プロジェクト概要
- **プロジェクト名**: JSMKC（Japan Super Mario Kart Championship 点数計算システム）
- **技術スタック**: Next.js 16, React 19, TypeScript, Tailwind CSS, NextAuth v5
- **生成日**: 2026-01-20

## テスト対象ページ一覧

### 公開ページ（認証不要）
| ページ | パス | 説明 |
|--------|------|------|
| トップページ | `/` | ランディングページ |
| サインインページ | `/auth/signin` | ログインページ |
| 認証エラーページ | `/auth/error` | 認証エラー表示 |

### 認証必要ページ
| ページ | パス | 説明 | ユーザータイプ |
|--------|------|------|---------------|
| プレイヤー管理 | `/players` | 選手一覧・管理 | admin |
| プロフィール | `/profile` | ユーザープロフィール | user |
| トーナメント一覧 | `/tournaments` | 大会一覧・管理 | admin |
| トーナメント詳細 | `/tournaments/[id]` | 大会詳細ハブ | user |

## 補足情報

### テスト環境
- **開発環境**: http://localhost:3000
- **本番環境**: 未設定

### 認証ユーザータイプ
- **admin**: Discord認証による管理者
- **user**: プレイヤー資格認証

### 注意事項
- 開発サーバーが起動している必要があります（`npm run dev`）
- 認証テストは認証状態ファイルが必要です
- 本番環境でのテストは公開ページのみ対象
