## 2. 管理者フロー

### 2.1 グループ設定

1. GP予選ページ → **「グループ設定」** ボタンをクリック
2. プレイヤーを選択し、グループに割り当て
3. **「グループ＆試合作成」** をクリック
4. 試合が自動生成（カップもランダム割当: §7.4）

### 2.2 スコア入力

1. **「試合一覧」** タブで試合を選択
2. 各レースの順位（1位/2位/ゲームオーバー）を入力
3. ドライバーズポイントが自動計算される
4. 1カップ = 5レース
5. **「結果保存」** をクリック

### 2.3 カップ代替ルール (§7.1)

- Star → Mushroom に代替可能
- Special → Flower に代替可能
- スコア入力ダイアログで切り替えボタンあり

### 2.4 ゲームオーバー対応 (§7.2)

- 対戦相手が5コース目前にゲームオーバーの場合、残ったプレイヤーは1人で完走
- 順位入力で「Game Over」（0）を選択 → ドライバーズポイント0

## 3. プレイヤーフロー

### 3.1 レース結果報告

1. GP予選ページの **「Enter Score」** → participant ページ
2. 未完了の自分の試合が表示
3. カップ情報が表示（カップ代替ボタンあり）
4. **「Add Race」** でレース追加（最大5レース）
5. コース選択（カップでフィルタ済み）+ 各プレイヤー順位入力（0-4）
6. ドライバーズポイントが自動表示
7. **「Submit Match Result」** をクリック

### 3.2 Dual Report

BM/MRと同じ二重報告システム。

## 4. 観覧者フロー

- GP予選ページで順位表を閲覧（ログイン不要）
- 順位: ドライバーズポイント（降順）→ 勝ち点（降順）

---

## 5. 決勝ステージ

### 5.1 ブラケット生成

- GP決勝ページ → **「Generate Bracket」**
- Top 8 / Top 16 を選択可能
- ダブルイリミネーション方式

## 6. スコアルール

| 項目 | 予選 | 決勝 |
|------|------|------|
| レース数/試合 | 5（1カップ） | 5（1カップ） |
| カップ | ランダム割当 | 主催者指定 |
| ポイント | 1位=9, 2位=6, 3位=3, 4位=1, GO=0 | 同左 |
| 勝利条件 | 合計ドライバーズポイント | ブラケット |
| BYE | 45-0 自動完了 | なし |
| 順位計算 | ドライバーズポイント → 勝ち点 → H2H | ブラケット |
| カップ代替 | Star→Mushroom, Special→Flower | 同左 |
