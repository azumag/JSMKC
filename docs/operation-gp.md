# グランプリ 操作マニュアル

このマニュアルは、グランプリ（GP）を大会スタッフが運用するための手順書です。GP はカップごとに 5 レースを行い、ドライバーズポイントの合計で試合結果を決めます。

---

## 1. グランプリの流れ

1. 管理者が参加者を選び、グループを作成します。
2. 予選はグループ内の総当たりで行います。
3. 各試合にはカップが割り当てられます。
4. 各レースの順位を入力すると、ポイントが自動計算されます。
5. 予選終了後、同順位を解決して「予選を確定」します。
6. 決勝ブラケットまたは Top 24 バラッジを開始します。

### 1.1 使うページ

| ページ | 用途 |
|--------|------|
| `/tournaments/[id]/gp` | 予選順位、試合一覧、管理操作 |
| `/tournaments/[id]/gp/participant` | プレイヤー本人の結果報告 |
| `/tournaments/[id]/gp/match/[matchId]` | 試合詳細の共有ページ |
| `/tournaments/[id]/gp/finals` | 決勝ブラケット |

---

## 2. 予選準備

### 2.1 グループを作成する

1. GP ページを開きます。
2. 「グループ設定」を押します。
3. 参加させるプレイヤーにチェックを入れます。
4. 必要に応じてシード番号を入力します。
5. グループ数を選びます。
6. 「シード順で振分け」を押すと、シードをもとに自動配置できます。
7. 内容を確認して保存します。

保存すると、グループ内の総当たり試合が自動で作成されます。GP 予選では、キノコ、フラワー、スター、スペシャルの 4 カップをランダムな順番に並べたものを 5 回作り、その並びから各ラウンドへ 1 カップずつ順番に割り当てます。同じラウンドの試合は、グループが違っても同じカップになります。デッキのつなぎ目では、前のラウンドと同じカップが続かないように調整されます。

### 2.2 グループをやり直す

「グループ編集」から既存設定を変更できます。ただし、試合が再作成される場合があり、入力済みの結果に影響します。

当日中にやり直す場合は、すでに案内済みの対戦や配信台に影響がないか確認してください。

---

## 3. 予選結果入力

### 3.1 入力ルール

| 項目 | 内容 |
|------|------|
| レース数 | 1 試合 5 レース |
| 入力方法 | 各レースで両プレイヤーの順位を入力 |
| 順位 | 1 位から 8 位 |
| 同順位 | 同じレースで両プレイヤーを同順位にはできません |
| ゲームオーバー | 管理者入力では 0 点扱いにできます |
| BREAK | 相手が BREAK の場合は自動で `45-0` |

### 3.2 ドライバーズポイント

| 順位 | ポイント |
|------|----------|
| 1 位 | 9 |
| 2 位 | 6 |
| 3 位 | 3 |
| 4 位 | 1 |
| 5 位から 8 位 | 0 |
| ゲームオーバー | 0 |

5 レースの合計ポイントが、その試合のスコアになります。

コースは割り当てられた 1 カップ内の 5 レースを走ります。各試合のカップ数は常に 1 です。

### 3.3 管理者が入力する

1. GP ページの「試合一覧」を開きます。
2. 対象試合の「結果入力」を押します。
3. カップを確認します。
4. 5 レース分のコースと両プレイヤーの順位を入力します。
5. 自動計算されたポイントを確認します。
6. 「結果保存」を押します。
7. 順位表に反映されたことを確認します。

合計ポイントだけを修正したい場合は、管理者用の手動修正欄を使います。レース単位の記録と合計ポイントがずれる可能性があるため、実際の記録を確認してから使ってください。

### 3.4 プレイヤーが報告する

プレイヤーは `/tournaments/[id]/gp/participant` から自分の未完了試合を確認し、結果を報告します。

二重報告が有効な大会では、両プレイヤーの報告が一致した時点で確定します。不一致の場合は、管理者が実際の結果を確認して修正します。

---

## 4. カップ代替

大会ルールにより、次の代替ができます。

| 割り当てカップ | 代替できるカップ |
|----------------|------------------|
| Star | Mushroom |
| Special | Flower |

代替を使う場合は、スコア入力画面でカップを切り替えてから結果を保存します。逆方向の代替はできません。

---

## 5. 試合一覧の見方

| 表示 | 意味 |
|------|------|
| `- - -` | まだ結果がありません |
| `31 - 22` | 確定済みのポイント |
| 報告欄 | プレイヤーからの報告状況 |
| Cup | 割り当てカップ |
| TV# | 配信台番号 |
| BREAK | 不戦勝の相手 |

GP は順位表でドライバーズポイントが重視されます。試合に勝ったかだけでなく、各レースのポイント差も確認してください。

---

## 6. 同順位が出た場合

順位表に同順位の警告が出たら、サドンデスプレーオフなど大会ルールに従って順位を決めます。

1. 警告が出ているグループを確認します。
2. 「プレーオフ結果を記録」を開きます。
3. プレイヤーを最終順位順に並べます。
4. 保存します。
5. 順位表の警告が消えたことを確認します。

保存した順位は、決勝進出者の並びに使われます。

---

## 7. 予選を確定する

全試合が終わり、順位が正しいことを確認したら「予選を確定」を押します。

予選確定後は、スコア編集がロックされます。修正が必要な場合は「予選のロック解除」を押し、修正後にもう一度「予選を確定」します。

---

## 8. 決勝を開始する

### 8.1 通常決勝

予選通過者が 16 名以下の場合は、「決勝ブラケットを生成」を押します。作成後は「トーナメントを見る」または「決勝へ」から決勝ページに移動します。

### 8.2 Top 24 バラッジ

予選通過者が 17 名以上の場合は、「バラッジ開始 (Top 24)」を押します。バラッジ結果を入力したあと、決勝ブラケットへ進みます。

### 8.3 決勝の試合形式

GP 決勝はダブルイリミネーション形式です。試合カードを開いてカップごとの結果を入力すると、勝者・敗者が次の試合へ進みます。

GP 決勝では、キノコ、フラワー、スター、スペシャルの 4 カップをランダムな順番に並べたものを 2 回作り、その並びを順番に走ります。

| 段階 | 目安 |
|------|------|
| バラッジ | 1 カップ勝負 |
| 決勝序盤から中盤 | 先に 2 カップ勝利 |
| Winners Final / Losers Final / Grand Final | 先に 3 カップ勝利 |

画面上の入力欄に従って、必要なカップ勝利数に達した結果を保存してください。

---

## 9. よくあるトラブル

### 同じ順位を入力してしまった

同じレースで両プレイヤーを同順位にはできません。実際の着順を確認して入力し直してください。

### カップを代替したい

Star は Mushroom、Special は Flower に代替できます。スコア入力画面で切り替えてから保存してください。

### 予選確定後に修正したい

「予選のロック解除」を押してから修正します。修正後は再度「予選を確定」してください。

### 決勝ボタンが出ない

予選が未確定、参加者が不足している、同順位警告が残っている、またはブラケットがすでに存在する可能性があります。

### 予選開始前

- [ ] カップ代替の扱いをスタッフ間で確認した
- [ ] 必要な試合に TV# を設定した

### 予選終了時

- [ ] 全試合が完了している
- [ ] 不一致報告が残っていない
- [ ] 同順位警告を解決した
- [ ] 順位表を確認した
- [ ] 予選を確定した

### 決勝開始時

- [ ] 通常決勝か Top 24 バラッジかを確認した
- [ ] ブラケットを生成した
- [ ] 決勝ページを開いて対戦カードを確認した
- [ ] 配信用 TV# を確認した
