# SMK 大会標準ルール

SMKコミュニティにおける大会標準ルール（大会主催者向け）

---

## 1. 概要

本ゲームは現在、多くの国で人気を博しています。ヨーロッパ、日本、アメリカなど、世界各国で選手権が開催されています。選手権のルールを標準化・定義することで、選手権の開催と参加を容易にすることを目的としています。

以下の4つのモードの競技が、SMKコミュニティの標準ルールとして定義されています。

- タイムアタック
- バトルゲーム
- VSマッチレース
- 150ccグランプリ

4つのモードそれぞれで、競技ごとにプレイヤーの順位が決定されます。予選ステージと決勝ステージの順位に応じてポイントが加算され、4つのモードの合計ポイントが総合ポイントとして算出されます。

---

## 2. モード共通の基本ルール

- 本大会は任天堂純正スーパーファミコン本体およびソフトを使用し、標準規格（NTSC/PAL）で実施されます。
- スーパーファミコンのコントローラーは、どの機種でも使用可能です。左右の方向キーの同時押しも含みます。
- 全4モードにおいて、予選ステージと決勝ステージを実施し、各モード内および全モード総合での順位を決定します。
- 使用中のテレビに不具合が発生した場合は、速やかに主催者にご連絡ください。
- SMK大会は、タイムトライアル、マッチレース、150cc GPにおいて、Sami Cetin氏が更新・管理する世界ランキングルールの非ショートカットカテゴリーに準じて開催されます。
  - https://mariokartplayers.com/smk/join.php

## 3. タイムアタックのルール

### 3.1 予選ステージ

- プレイヤーはマリオサーキット1からレインボーロードまでの全20コースを順番に走行し、各コースのタイムを記録します。
- 全員が走行を終えた後、全タイムを集計し、各コースの順位に基づいてプレイヤーごとのポイントが算出されます。
- 全20コースの合計ポイントで予選順位を決定し、上位24名が決勝進出となります。
- 合計ポイントが同点の場合、全コースの合計タイムが速いプレイヤーが上位となります。

#### ペア走行

- 予選ステージではペアを組み、1組につき1台のテレビを割り当てます。
- ペアは主催者により、TA経験豊富なプレイヤーと経験の浅いプレイヤーが1人ずつになるよう編成されます。
- 1人が走行している間、もう1人が相手のタイムを記録します。1人が全20コースを完了後、役割を交代します。
- 原則としてTA経験の少ないプレイヤーが先にプレイするが、ペア間で協議の上、順番を決定してもよいです。

### 3.2 決勝ステージ

決勝ステージは3つのフェーズに分かれています。

#### フェーズ 1

- 予選ステージ17位から24位までの8名のプレイヤーによるバトルロワイヤルが開催されます。
- 競技はランダムに選出された1つのコースで同時に行われ、コースで最も遅いタイムを記録したプレイヤーが敗退します。
- フェーズ1では、このプロセスを8名から残り4名になるまで繰り返します。

#### フェーズ 2

- フェーズ1の通過者4名と予選ステージ通過者13位から16位までの計8名によるバトルロワイヤルが開催されます。
- フェーズ1と同様に、ランダムに選出された1つのコースで同時に行われ、レースごとに1名が敗退し、8名から残り4名になるまで繰り返します。

#### フェーズ 3

- フェーズ2を通過した4名のプレイヤーと予選通過者上位12名の計16名によるバトルロワイヤルが開催されます。
- フェーズ3では、16名のプレイヤーそれぞれに3つのライフが与えられます。ライフをすべて失ったプレイヤーは敗退となります。
- 競技は1コースずつ進行します。これまでのフェーズとは異なり、遅いタイムを記録した下位半分のプレイヤーはライフを1つ失います。
- このプロセスを繰り返し、ライフが0になったプレイヤーは敗退し、16名から8名になるまで続きます。
- 残った8名のプレイヤーのライフは3にリセットされ、コースごとに競技が再開されます。
- 上位4名になった時、上位2名になった時もライフは3にリセットされ、最後の1名になるまで競技が続けられます。
- 残った最後の1名が勝者となります。

### 3.3 タイムアタック注意事項

- 競技は全て1コースにつき1回のみの実施となります。ポーズやリトライは禁止です。リトライが発生した場合、そのコースのタイムは9分59秒99となります。
- ゴーストやCPU対戦相手が表示されている間は走らないでください。
- ラップスキップ（ゴール付近でのバグによるショートカット）は禁止です。
- 予選ステージおよび決勝ステージの試合中は、練習を控えてください。
- 各フェーズ開始時に、プレイヤーにテレビが割り当てられます。
- 試合開始後は、休憩の合図または試合終了まで、使用しているテレビの前にいてください。
- 予選ステージに3人以上のプレイヤーが残っている場合は、ロケットスタートのシグナルを聞き逃さないよう、スタート時間をずらすように促してください。
- 決勝ステージではランダムにコースが選択され、20コースすべてが選択されるまで同じコースは走りません。これは各フェーズで適用されます。
- フェーズ2の開始前、およびトップ16、トップ8、トップ4、トップ2決定戦の開始前に、5〜10分の短い休憩があります。
- 主催者は各プレイヤーに使用するテレビの番号をアナウンスします。

## 4. 2Pモードのルール

### 4.1 予選ステージ共通

- プレイヤーは試合中にキャラクターを変更できます。2人プレイの場合は、プレイヤー間でキャラクターの選択について相談してください。
- 1Pと2Pの割り当て方については主催者が決定します。主催者から特に指示がない限り、試合中は割り当てをそのままにしてください。
- TV1またはTV2に割り当てられた試合は、指定されたTVでプレイしてください。TVの割り当てがない場合、TV1またはTV2以外のどのTVでも構いません。対戦相手と相談して決めてください。
- 参加プレイヤーは参加人数に基づいて2〜4のグループに分かれ、各グループで総当たり戦を行います。グループは主催者がレベルバランスを考慮して割り当てます。予選順位は全グループから合算し、上位24名が進出します。
- 対戦カードが配布されますので、カードの上から順に試合を進めてください。
- 対戦相手がBreak（休憩）とマークされている場合、不戦勝扱いとなります。VSマッチレースとバトルゲームは4-0、GP150ccは45-0となります。
- 予選ステージでは、バトルゲームとVSマッチレースはそれぞれ4試合、GP150ccはカップ戦1試合で行われます。

#### 予選順位決定基準

**バトルモード / マッチレース:**
1. ポイント
2. 勝敗数
3. 直接対決結果
4. サドンデスプレーオフ

**150cc GP:**
1. ドライバーズポイント
2. 勝敗数
3. 直接対決結果
4. サドンデスプレーオフ

> ※GPについては、上位プレイヤーが新規プレイヤーのポイントを失わせようと躍起になるのを防ぐため、CDM2024よりこのルールを変更しました。

#### ポイント計算

- 勝利 = 2ポイント
- 引き分け = 1ポイント
- 敗北 = 0ポイント

例）5試合：3勝1引き分け1敗 = 7ポイント

#### 同順位の解決

1. **ポイント**: 上記のポイント計算による
2. **勝敗数**: ポイント差による勝敗数
3. **直接対決**: 選手Aと選手Bが①と②で同じ結果になった場合、選手Aと選手Bの対戦時の勝者が上位となります
4. **サドンデスプレーオフ**: ③で比較した対戦相手同士でサドンデスマッチが行われます

### 4.2 決勝ステージ共通

- 予選リーグ1位から12位までの選手はトップ16への出場権を獲得しています。そのため、まず予選リーグ12位から24位までの12名によるトーナメントを実施します。
- このトーナメントで12名から勝者4名を決定します。
- トップ16に進出した選手は、2敗で敗退となるダブルイリミネーション方式のトーナメントで競います。
- 決勝ステージでは、各ステージで必要な勝利数、使用するコース・カップ、使用するTVが定められています。これらの詳細については主催者からのアナウンスに従ってください。

## 5. バトルゲームのルール

2Pモードのルール（4.1: 予選ステージ共通、4.2: 決勝ステージ共通）に加え、バトルゲームでは以下のルールが適用されます。

### 5.1 テレサの使用

- テレサを入手したら、なるべく早くテレサを使って他のアイテムを拾いましょう。また、相手にテレサを使われてアイテムを取られないようにしましょう。
- ただし、相手のスターが有効な間はテレサをキープできます。その場合は相手のスターが切れたら、すぐにテレサを使用してください。

### 5.2 コース固有ルール

- コース2のブロック内のプールに入った場合は、10秒以内に出てください。

### 5.3 キャラクター選択

- 各コース終了後、そのコースで負けたプレイヤーにキャラクター選択権が与えられます。
- この場合、そのコースで勝ったプレイヤーが先にキャラクターを選択してください。
- ※各コース終了後、両方のプレイヤー（前のコースの勝者と敗者）はキャラクター選択を希望できます。希望する場合はプレイヤー間で相談してください。

### 5.4 予選・決勝の進行

- 予選ステージは対戦相手毎に4コースで構成されます。コース1からコース4までを順番に使用して対戦してください。
- 試合の進行が遅い場合、主催者が小さなコース1を指示する場合があります。
- 決勝ステージもコース1からコース4までを順番に使用して対戦しますが、順番の開始コースが変わりますので、開始コースについては主催者のアナウンスに従ってください。

## 6. VSマッチレースのルール

2Pモードのルール（4.1: 予選ステージ共通、4.2: 決勝ステージ共通）に加え、VSマッチレースでは以下のルールが適用されます。

### 6.1 禁止事項

- ラップスキップ（ゴール付近でのバグを使ったショートカット）は禁止です。

### 6.2 ショートカット

- キノコや羽根を使ったショートカットは、基本的に全コースで許可されています。
- しかし、コースによっては特殊なショートカットが禁止されています。

#### コース別ショートカットルール

| コース | ルール |
|--------|--------|
| ノコノコビーチ1 | 1コーナーでのショートカットを利用した空中ゴールは禁止 |
| レインボーロード | 最終ラップのみ空中ゴールが許可。最終ラップ以外での空中ゴールは禁止 |

- これらのルールに違反した場合は、即失格となります。

### 6.3 予選・決勝の進行

- 予選ステージは各4コースずつで行われます。対戦カードに記載されているコースを使用してください。
- 決勝ステージでは、各試合で使用するコース、およびコース数が主催者からアナウンスされます。

---

## 7. 150cc グランプリのルール

2Pモードのルール（4.1: 予選ステージ共通、4.2: 決勝ステージ共通）に加え、150ccグランプリでは以下のルールが適用されます。

### 7.1 カップ選択

- 難しいカップをクリアできないプレイヤーへの対策として、スターカップの代わりにキノコカップ、スペシャルカップの代わりにフラワーカップの選択が許されています。プレイヤー間で相談してください。
- ※その他、主催者の判断により指示が出る場合もあります。

### 7.2 ゲーム進行

- カップ1つが終了し、ドライバーズポイントが記録されるまで、本体をリセットしないでください。
- どちらのプレイヤーにも残機がない場合は、両プレイヤーがゲームオーバーになった時点でドライバーズポイントを記録してください。
- 対戦相手がレースを完走するまでリタイアしないでください。
- 対戦相手が5コース目前にゲームオーバーになった場合、残ったプレイヤーは5コース目を1人で完走しなければなりません。

### 7.3 ショートカット

- 周回スキップ、特殊ショートカットについてはVSマッチレースのルールと同様です。
- ただし、レインボーロードでは、最終ラップのみゴールラインフィニッシュが可能です。
- VSマッチレースとは異なり、最終ラップ以外では、羽を使って上空を超え、着陸するショートカットが許可されています（ゴールライン直前を含む）。
- ただし、空中で落下した場合は失格となりますので、ご注意ください。

### 7.4 予選・決勝の進行

- 予選ステージは1カップ戦となります。対戦カードに記載されているカップを選択してください。
- 決勝ステージでは、各試合で使用するカップ、およびカップ数が主催者からアナウンスされます。

## 8. 総合順位

### 8.1 ポイント計算概要

- 総合順位は、プレイヤー毎に各モードの成績に基づいて算出されます。
- 各モードでは、予選ステージで最大1000ポイント、決勝ステージで最大2000ポイントを獲得できます。
- 4つのモード全体では最大12000ポイントを獲得できます。

### 8.2 タイムアタック予選ステージのポイント

- コース毎に最速タイムを記録したプレイヤーには50ポイントが付与され、最も遅いタイムを記録したプレイヤーには0ポイントが付与されます。
- 参加人数に応じて線形補間されたポイントが記載されたスコアテーブルが作成されます。
- 各プレイヤーは順位に基づいて、このスコアテーブルに対応したポイントを獲得します。

**例：11人が参加し、コースMC1で2位だった場合**

| 10位 | 5 |
| 11位 | 0 |

- 各プレイヤーの予選ステージの総合スコアは、各コースのスコアの合計です。
- 総合スコアの小数点以下は切り捨てられます。

### 8.3 2P対戦予選ステージのポイント

- 全試合に勝利すると1000ポイント獲得し、全試合に敗北すると0ポイントとなります。
- 全試合数に応じて線形補間されたポイントが記載されたスコアテーブルが作成されます。
- 各プレイヤーは獲得ポイント数に基づいて、このスコアテーブルに対応したポイントを獲得します。
- 小数ポイントは四捨五入されます。

## 9. 用語集

| 用語 | 説明 |
|------|------|
| SMK | 本システムで対象とする競技用レースゲーム |
| NTSC | 日本・北米で使用される映像規格 |
| PAL | ヨーロッパで使用される映像規格 |
| ダブルイリミネーション | 2回負けると敗退するトーナメント形式 |
| バトルロワイヤル | 複数人が同時に競い、下位が脱落する形式 |
| サドンデス | 即座に勝敗が決まる延長戦 |
| ラップスキップ | ゴール付近のバグを利用したショートカット |
| ドライバーズポイント | GPモードでの順位に応じた得点（9, 6, 3, 1） |
| テレサ | 相手のアイテムを奪うアイテム |
