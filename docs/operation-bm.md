## 1. 概要

### 画面構成

| 画面 | パス | 用途 |
|------|------|------|
| BM予選 | `/tournaments/[id]/bm` | 予選グループステージ管理 |
| BM決勝 | `/tournaments/[id]/bm/finals` | ダブルイリミネーション決勝 |
| スコア入力 | `/tournaments/[id]/bm/participant` | プレイヤー自己報告 |
| 試合詳細 | `/tournaments/[id]/bm/match/[matchId]` | 個別試合の詳細 |

### スコアルール (§5)
- 予選: 4ラウンド制（score1 + score2 = 4）、2-2引き分けあり（§4.1: draw = 1ポイント）
- 決勝: Best-of-9（先に5勝した方が勝利）
- バトルコース1〜4を順番に使用

## 2. 管理者フロー

### 2.1 グループ設定

1. BM予選ページ → **「グループ設定」** ボタンをクリック
2. ダイアログでプレイヤーを選択（チェックボックス）
3. グループ数を選択（2/3/4）
4. **「シード順で振り分け」** ボタンでシード順に自動配置（任意）
5. **「グループ＆試合作成」** をクリック
6. ラウンドロビン方式の試合が自動生成される

### 2.2 スコア入力

1. **「試合一覧」** タブで試合を選択
2. 各プレイヤーのスコアを入力（0-4、合計=4）
3. **「スコア保存」** をクリック
4. 順位表が自動更新される

### 2.3 グループ編集

- **「グループ編集」** ボタンで既存のグループを変更可能
- 変更すると既存の試合結果が再生成される（警告表示）

### 2.4 データエクスポート

- **「Export CSV」** ボタンで順位表・試合結果をCSVダウンロード

## 3. プレイヤーフロー

### 3.1 スコア報告

1. ログイン後、BM予選ページの **「Enter Score」** ボタンをクリック
2. 自分の対戦（未完了のもの）が表示される
3. 各プレイヤーの勝利ラウンド数を入力（0-5）
4. **「Submit Scores」** をクリック
5. 対戦相手も同じスコアを報告すると自動確定

### 3.2 Dual Report（二重報告）

- 両プレイヤーが同じスコアを報告 → 自動確定
- スコアが一致しない → 管理者に通知、手動確認が必要
- 片方のみ報告 → もう一方の報告待ち

### 3.3 キャラクター選択優先権 (§5.3)

スコア入力ダイアログに「前ラウンドで負けたプレイヤーが先にキャラクターを選択」のガイダンスが表示されます。

## 4. 観覧者フロー

- BM予選ページにアクセス → 順位表・試合結果を閲覧
- グループ未設定時は「セットアップが完了するまでお待ちください」と表示
- ログイン不要

## 5. 決勝ステージ

### 5.1 ブラケット生成

1. BM決勝ページ → **「Generate Bracket」** ボタン
2. ブラケットサイズを選択: **Top 8** or **Top 16**
3. 予選順位に基づいてシード配置
4. ダブルイリミネーション方式（2敗で脱落）

### 5.2 ブラケット構造

**8人ブラケット (17試合)**:
- Winners: QF(4) → SF(2) → Final(1)
- Losers: R1(2) → R2(2) → R3(2) → SF(1) → Final(1)
- Grand Final + Reset

**16人ブラケット (31試合)** (§4.2):
- Winners: R1(8) → QF(4) → SF(2) → Final(1)
- Losers: L_R1(4) → L_R2(4) → L_R3(2) → L_R4(2) → SF(1) → Final(1)
- Grand Final + Reset

### 5.3 決勝スコア入力

- Best-of-9: 各プレイヤー最大5勝（先に5勝で勝利）
- 開始コース: BC1〜BC4のガイダンス表示

---

## 6. スコアルール

| 項目 | 予選 | 決勝 |
|------|------|------|
| ラウンド数 | 4 | Best-of-9 |
| 最大スコア | 4 | 5 |
| 勝利条件 | score > 2 | score >= 5 |
| BYE | 4-0 自動完了 | なし |
| 順位計算 | 勝ち点(勝×2+引×1) → 得失差 → H2H | ブラケット |
