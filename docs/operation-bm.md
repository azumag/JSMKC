# バトルモード 操作マニュアル

このマニュアルは、バトルモード（BM）を大会スタッフが運用するための手順書です。予選グループの作成、スコア入力、同順位の解決、決勝ブラケットの開始までを扱います。

---

## 1. バトルモードの流れ

1. 管理者が参加者を選び、グループを作成します。
2. 予選はグループ内の総当たりで行います。
3. 各試合は 4 ラウンド固定です。
4. 試合結果を入力すると、順位表が自動更新されます。
5. 予選終了後、同順位を解決して「予選を確定」します。
6. 決勝ブラケットまたは Top 24 バラッジを開始します。

### 1.1 使うページ

| ページ | 用途 |
|--------|------|
| `/tournaments/[id]/bm` | 予選順位、試合一覧、管理操作 |
| `/tournaments/[id]/bm/participant` | プレイヤー本人のスコア報告 |
| `/tournaments/[id]/bm/match/[matchId]` | 試合詳細の共有ページ |
| `/tournaments/[id]/bm/finals` | 決勝ブラケット |

---

## 2. 予選準備

### 2.1 グループを作成する

1. BM ページを開きます。
2. 「グループ設定」を押します。
3. 参加させるプレイヤーにチェックを入れます。
4. 必要に応じてシード番号を入力します。
5. グループ数を選びます。
6. 「シード順で振分け」を押すと、シードをもとに自動配置できます。
7. 内容を確認して保存します。

保存すると、グループ内の総当たり試合が自動で作成されます。

### 2.2 グループをやり直す

「グループ編集」から既存設定を変更できます。ただし、試合が再作成される場合があり、入力済みの結果に影響します。

当日中にやり直す場合は、必ず次を確認してください。

- すでに入力済みの試合がないか
- プレイヤーに案内済みの試合が変わってもよいか
- 配信台や進行表に影響がないか

---

## 3. 予選スコア入力

### 3.1 入力ルール

| 項目 | 内容 |
|------|------|
| ラウンド数 | 4 ラウンド固定 |
| 入力例 | `4-0`, `3-1`, `2-2`, `1-3`, `0-4` |
| 引き分け | `2-2` は有効 |
| クリア | 管理者は `0-0` で試合結果を消すことができます |
| BREAK | 相手が BREAK の場合は自動で `4-0` |

勝ち点は、勝ち 2 点、引き分け 1 点、負け 0 点です。順位表では勝ち点、得失差などをもとに順位が表示されます。

### 3.2 管理者が入力する

1. BM ページの「試合一覧」を開きます。
2. 対象試合の「スコア入力」を押します。
3. 両プレイヤーの勝利ラウンド数を入力します。
4. 「スコア保存」を押します。
5. 順位表に反映されたことを確認します。

入力後、試合一覧では完了済みの試合にスコアが表示されます。

### 3.3 プレイヤーが報告する

プレイヤーは `/tournaments/[id]/bm/participant` から自分の未完了試合を確認し、スコアを報告します。

二重報告が有効な大会では、両プレイヤーの報告が一致した時点で確定します。不一致の場合は、管理者が実際の結果を確認して修正します。

---

## 4. 試合一覧の見方

| 表示 | 意味 |
|------|------|
| `- - -` | まだ結果がありません |
| `3 - 1` | 確定済みの結果 |
| 青い報告表示 | 片方のプレイヤーが報告済み |
| 黄色い報告表示 | 両者が報告したが内容が不一致 |
| TV# | 配信台番号 |
| BREAK | 不戦勝の相手 |

TV# は管理者が選択できます。配信担当と共有している台番号に合わせて設定してください。

---

## 5. 同順位が出た場合

順位表に同順位の警告が出たら、サドンデスプレーオフなど大会ルールに従って順位を決めます。

1. 警告が出ているグループを確認します。
2. 「プレーオフ結果を記録」を開きます。
3. プレイヤーを最終順位順に並べます。
4. 保存します。
5. 順位表の警告が消えたことを確認します。

保存した順位は、決勝進出者の並びに使われます。

---

## 6. 予選を確定する

全試合が終わり、順位が正しいことを確認したら「予選を確定」を押します。

予選確定後は、スコア編集がロックされます。決勝ブラケットを作る前の最終確認として使ってください。

修正が必要な場合は「予選のロック解除」を押し、修正後にもう一度「予選を確定」します。

---

## 7. 決勝を開始する

### 7.1 通常決勝

予選通過者が 16 名以下の場合は、「決勝ブラケットを生成」を押します。作成後は「トーナメントを見る」または「決勝へ」から決勝ページに移動します。

### 7.2 Top 24 バラッジ

予選通過者が 17 名以上の場合は、「バラッジ開始 (Top 24)」を押します。バラッジ結果を入力したあと、決勝ブラケットへ進みます。

### 7.3 決勝の試合形式

BM 決勝はダブルイリミネーション形式です。試合カードを開いてスコアを入力すると、勝者・敗者が次の試合へ進みます。

| 段階 | 目安 |
|------|------|
| バラッジ | 先に 3 勝、後半は先に 4 勝 |
| 決勝序盤 | 先に 5 勝 |
| 準決勝以降 | 先に 7 勝 |

画面上の入力欄に従って、必要な勝利数に達したスコアを保存してください。

---

## 8. よくあるトラブル

### 予選確定ボタンを押したあとに修正したい

「予選のロック解除」を押してから修正します。修正後は再度「予選を確定」してください。

### 決勝ボタンが出ない

予選が未確定、参加者が不足している、同順位警告が残っている、またはブラケットがすでに存在する可能性があります。

### スコアが保存できない

BM 予選は通常 `4` ラウンド合計で入力します。`3-2` のように合計が 4 にならない結果は保存できません。結果を消したい場合は管理者が `0-0` を使います。

### 報告が不一致になった

プレイヤー双方に確認し、管理者が正しいスコアを保存します。

## 9. チェックリスト

### 予選開始前

- [ ] BM 参加者を設定した
- [ ] グループ数とシードを確認した
- [ ] 試合一覧が作成されている
- [ ] 必要な試合に TV# を設定した
- [ ] BM を公開するタイミングを決めた

### 予選終了時

- [ ] 全試合が完了している
- [ ] 不一致報告が残っていない
- [ ] 同順位警告を解決した
- [ ] 順位表を確認した
- [ ] 予選を確定した

### 決勝開始時

- [ ] 通常決勝か Top 24 バラッジかを確認した
- [ ] ブラケットを生成した
- [ ] 決勝ページを開いて対戦カードを確認した
- [ ] 配信用 TV# を確認した
