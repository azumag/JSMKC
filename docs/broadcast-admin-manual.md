# 配信管理者マニュアル

このマニュアルは、JSMKC スコアシステムを使って配信表示を管理する担当者向けの手順書です。OBS などの配信ソフトに表示するオーバーレイ、配信用 TV#、1P/2P 表示名、試合情報の反映を扱います。

---

## 1. 配信管理の全体像

JSMKC の配信向け機能は、次の 3 つに分かれます。

| 機能 | 用途 |
|------|------|
| TV# | どの配信台で行う試合かを、試合一覧や決勝ブラケット上で示す |
| 配信に反映 | 選択中の試合やプレイヤーを OBS 用表示へ送る |
| オーバーレイ | OBS のブラウザソースとして、名前・試合情報・進行イベントを表示する |

配信担当は、進行担当が決めた配信台の運用に合わせて TV# を確認し、配信中の試合を「配信に反映」します。

---

## 2. 使うページ

| ページ | 用途 |
|--------|------|
| `/tournaments/[id]` | 大会トップ、各競技への入口 |
| `/tournaments/[id]/broadcast` | 1P/2P の表示名を直接設定する配信管理ページ |
| `/tournaments/[id]/overlay/dashboard` | OBS に入れる常設ダッシュボード表示 |
| `/tournaments/[id]/overlay` | OBS に入れる速報トースト表示 |
| `/tournaments/[id]/ta` | TA 予選の TV# と配信反映 |
| `/tournaments/[id]/ta/phase1` | TA フェーズ1の TV# と配信反映 |
| `/tournaments/[id]/ta/phase2` | TA フェーズ2の TV# と配信反映 |
| `/tournaments/[id]/ta/finals` | TA 決勝の TV# と配信反映 |
| `/tournaments/[id]/bm` | BM 予選の TV# |
| `/tournaments/[id]/bm/finals` | BM 決勝の TV# と配信反映 |
| `/tournaments/[id]/mr` | MR 予選の TV# |
| `/tournaments/[id]/mr/finals` | MR 決勝の TV# と配信反映 |
| `/tournaments/[id]/gp` | GP 予選の TV# |
| `/tournaments/[id]/gp/finals` | GP 決勝の TV# と配信反映 |

`[id]` には大会ごとの ID またはカスタム URL が入ります。OBS に登録する URL は、本番で使う大会 URL と同じ `[id]` を使ってください。

---

## 3. 権限と URL の扱い

### 3.1 管理操作

TV# の変更や「配信に反映」は管理者ログインが必要です。

1. 画面右上の「ログイン」を開きます。
2. 管理者アカウントでログインします。
3. 対象大会の各競技ページ、または配信管理ページを開きます。

管理者用のボタンや TV# の選択欄が見えない場合は、管理者としてログインできているか確認してください。

### 3.2 OBS 用 URL

`/overlay` と `/overlay/dashboard` は、OBS のブラウザソースから読めるように認証なしで表示できます。URL を知っている人は表示できるため、配信関係者以外に共有しないでください。

---

## 4. OBS の設定

### 4.1 常設ダッシュボードを追加する

常設ダッシュボードは、右側の進行ログ、下部のフェーズ表示、1P/2P 名、現在の勝利数などを表示します。

1. OBS で対象シーンを開きます。
2. ソースを追加し、「ブラウザ」を選びます。
3. URL に `/tournaments/[id]/overlay/dashboard` を入力します。
4. 幅を `1920`、高さを `1080` にします。
5. 背景が透過されることを確認します。
6. 実際の配信レイアウトに合わせて、必要ならソースの表示位置を調整します。

### 4.2 速報トーストを追加する

速報トーストは、スコア入力や順位更新などのイベントを一時的に表示します。常設ダッシュボードと併用できます。

1. OBS で対象シーンを開きます。
2. ソースを追加し、「ブラウザ」を選びます。
3. URL に `/tournaments/[id]/overlay` を入力します。
4. 幅を `1920`、高さを `1080` にします。
5. 背景が透過されることを確認します。

### 4.3 OBS 側で確認すること

- [ ] 大会 ID またはカスタム URL が正しい
- [ ] 幅 `1920`、高さ `1080` になっている
- [ ] ブラウザソースの背景が透過されている
- [ ] 配信画面の選手名欄と 1P/2P 表示位置が合っている
- [ ] 右側の進行ログが配信画面の他要素と重なっていない

---

## 5. TV# の使い方

TV# は、試合やプレイヤーをどの配信台に割り当てるかを示す番号です。

| TV# | 用途 |
|-----|------|
| TV1 | TA では 1P 側の配信表示に反映される。2P 対戦モードでは、TV1 に割り当てた試合の 1P/2P が配信表示に反映される |
| TV2 | TA では 2P 側の配信表示に反映される。2P 対戦モードでは別配信台・記録用として扱う |
| TV3/TV4 | 進行上の控え・別配信台・記録用として使えるが、配信表示には反映されない場合がある |

TV# は、予選の試合一覧、決勝ブラケット、TA のラウンド入力欄などで設定できます。TA では TV1 と TV2 がそれぞれ配信表示の 1P/2P 名に対応します。BM/MR/GP などの 2P 対戦モードでは、TV1 の試合に含まれる 1P/2P が配信表示へ反映されます。大会側で TV2 以降を別配信台や記録用に使う場合は、当日の役割を事前に決めてください。

---

## 6. 配信に反映する

### 6.1 試合から反映する

BM/MR/GP の決勝や一部の試合画面では、対象試合の「配信に反映」を押すと、次の情報がオーバーレイに送られます。

- 1P/2P の表示名
- 試合ラベル
- 現在の勝利数
- First To の勝利数

反映後、OBS の `/overlay/dashboard` に数秒以内に表示されることを確認してください。

### 6.2 TA から反映する

TA では、アクティブなプレイヤーに TV# を割り当ててから「配信に反映」を押します。

1. TA の対象ページを開きます。
2. 配信したいプレイヤーに TV1 または TV2 を割り当てます。
3. 「配信に反映」を押します。
4. OBS の 1P/2P 表示名が更新されたことを確認します。

TV3/TV4 を割り当てたプレイヤーは、記録上は TV# として残りますが、1P/2P 表示名には送られません。

### 6.3 配信管理ページから直接反映する

試合一覧や TA 画面を使わずに、表示名だけを直接変えたい場合は `/tournaments/[id]/broadcast` を使います。

1. 管理者で `/tournaments/[id]/broadcast` を開きます。
2. 「現在の配信表示」で OBS に出ている名前を確認します。
3. 1P 名前、2P 名前をプレイヤーリストから選ぶか直接入力します。
4. 「配信に反映」を押します。
5. OBS の表示が更新されたことを確認します。

表示を消したい場合は「クリア」を押します。

---

## 7. 当日の運用手順

### 7.1 大会開始前

1. 管理者でログインします。
2. 大会トップから各競技ページを開けることを確認します。
3. OBS に `/overlay/dashboard` と必要に応じて `/overlay` を登録します。
4. テスト用に `/broadcast` から 1P/2P 名を設定します。
5. OBS 側で名前が表示されることを確認します。
6. 確認後、必要なら「クリア」で表示を消します。

### 7.2 予選中

1. 配信する試合の TV# を確認します。
2. 必要に応じて TV# を設定または修正します。
3. スコア入力担当と、次に配信する試合を確認します。
4. TA では TV1/TV2 を割り当てて「配信に反映」を押します。
5. BM/MR/GP では、決勝や反映ボタンがある画面で「配信に反映」を押します。
6. OBS の表示が更新されていることを確認します。

### 7.3 決勝中

1. 決勝ブラケットで配信対象の試合を確認します。
2. 該当試合に TV# を設定します。
3. 対象試合の「配信に反映」を押します。
4. 1P/2P 名、試合ラベル、スコア表示が合っているか確認します。
5. 試合が終わったら結果を保存し、次の試合に移る前に表示を更新します。

## 8. よくあるトラブル

### OBS に何も表示されない

URL の大会 ID が正しいか、ブラウザソースの幅と高さが `1920 x 1080` か確認してください。大会ページをブラウザで直接開き、`/overlay/dashboard` が表示できるかも確認します。

### 名前が更新されない

管理画面で「配信に反映」を押したか確認してください。OBS 側のブラウザソースを更新すると反映される場合があります。

### TV2/TV3/TV4 の選手名が出ない

TA では TV1/TV2 が 1P/2P 名として反映されます。BM/MR/GP などの 2P 対戦モードでは TV1 の試合の 1P/2P が反映されるため、TV2/TV3/TV4 に割り当てた試合は OBS の 1P/2P 表示へ出ない場合があります。配信表示へ出したい試合は TV1 に割り当て直してください。

### 管理画面で TV# を変更できない

管理者としてログインしているか確認してください。ログイン済みでも表示されない場合は、ページを更新して再確認します。

### 別の試合の名前が残っている

次の試合で「配信に反映」を押すか、`/broadcast` の「クリア」で一度表示を消してください。

### 配信画面と実際の 1P/2P が逆になっている

TA の場合は各プレイヤーの TV# を見直してください。BM/MR/GP のような 2P 対戦モードでは、TV1 に割り当てた対象試合の表示順を確認してから再度「配信に反映」を押します。
