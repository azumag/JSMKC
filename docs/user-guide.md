## 1. ログイン

### プレイヤーログイン

1. サイト右上の **「ログイン」** をクリック
2. **「プレイヤー」** タブを選択
3. 管理者から受け取った **ニックネーム** と **パスワード** を入力
4. **「ログイン」** ボタンをクリック

> パスワードは大会管理者がプレイヤー登録時に発行します。紛失した場合は管理者に再発行を依頼してください。

### ログイン後

- ヘッダーにあなたのニックネームが表示されます
- スコア入力ページにアクセスできるようになります

## 2. 大会ページの見方

### トップページ → トーナメント → モード選択

1. **「トーナメント」** メニューから大会を選択
2. タブで各モードに切り替え:
   - **タイムトライアル** — 個人タイムアタック
   - **バトルモード** — 1対1 バルーンバトル
   - **マッチレース** — 1対1 レース対戦
   - **グランプリ** — カップ別ドライバーズポイント
   - **総合** — 全4モードの合計ランキング

### 順位表の見方

各モードページに順位表が表示されます：
- **MP**: 試合数
- **W / T / L**: 勝ち / 引き分け / 負け
- **得点**: 勝ち点（勝×2 + 引×1）

## 3. バトルモード（BM）のスコア入力

### 手順

1. BM予選ページの **「Enter Score」** ボタンをクリック
2. あなたの未完了の対戦が表示されます
3. 各プレイヤーの **勝利ラウンド数**（0〜5）を入力
4. **「Submit Scores」** をクリック

### ルール
- 予選: 4ラウンド制（合計=4、引き分けなし）
- 決勝: Best-of-9（先に5勝で勝利）
- 前ラウンドで負けた方がキャラクター選択の優先権を持ちます

---

## 4. マッチレース（MR）のスコア入力

### 手順

1. MR予選ページの **「Enter Score」** ボタンをクリック
2. **「Add Race」** ボタンでレースを追加（最大5レース）
3. 各レースで:
   - **コース** を選択
   - 各プレイヤーの **順位**（1位/2位）を入力
4. **「Submit Match Result」** をクリック

### ルール
- 5レース対戦
- 各レースの1位がそのレースの勝者
- 5レース中の勝ち数で試合の勝敗が決定

---

## 5. グランプリ（GP）のスコア入力

### 手順

1. GP予選ページの **「Enter Score」** ボタンをクリック
2. カップ情報が表示されます（代替カップへの切り替えも可能）
3. **「Add Race」** でレースを追加（1カップ = 5レース）
4. 各レースで:
   - **コース** を選択（カップ内のコースのみ表示）
   - 各プレイヤーの **順位**（0〜4）を入力
   - **0 = ゲームオーバー**（ドライバーズポイント0）
5. ドライバーズポイントが自動計算されます
6. **「Submit Match Result」** をクリック

### ドライバーズポイント

| 順位 | ポイント |
|------|---------|
| 1位 | 9 |
| 2位 | 6 |
| 3位 | 3 |
| 4位 | 1 |
| ゲームオーバー | 0 |

### カップ代替ルール
- Star → Mushroom に変更可能
- Special → Flower に変更可能

## 6. タイムトライアル（TA）のタイム入力

### 手順

1. TA予選ページの **「Enter Score」** ボタンをクリック
2. 各コースのタイムを入力（形式: `M:SS.mmm`、例: `1:23.456`）
3. **「Submit Times」** をクリック

### タイム入力のヒント

- 数字のみ入力しても自動でフォーマットされます
  - `58490` → `0:58.490`
  - `123` → `1:23.000`
- 20コース全てのタイムを入力すると合計タイムが計算されます

### パートナー制（ペア走行）

管理者がパートナーを設定した場合:
- あなたのパートナーがあなたのタイムを入力します
- あなたはパートナーのタイムを入力します
- 自分のタイムは直接編集できません

## 7. スコアの確認方法

### 順位表
各モードページの **「順位表」** タブで現在の順位を確認できます。

### 総合ランキング
**「総合」** タブで全4モードの合計ポイントによるランキングを確認できます。

### CSV出力
管理者が **「Export CSV」** ボタンで結果をダウンロードできます。

## 8. 二重報告（Dual Report）について

BM/MR/GPでは **二重報告システム** を採用しています：

1. あなたがスコアを報告します
2. 対戦相手も同じスコアを報告します
3. **両方のスコアが一致** → 自動的に確定されます
4. **スコアが不一致** → 管理者に通知され、手動確認が行われます

> 片方のみ報告した場合は、もう一方の報告を待つ状態になります。

## 9. よくある質問

### Q: パスワードを忘れました
管理者に連絡してパスワードの再発行を依頼してください。

### Q: スコアを間違えて入力しました
管理者に連絡して修正を依頼してください。プレイヤーからの修正はできません。

### Q: 対戦相手と報告したスコアが違います
管理者が確認し、正しいスコアを入力します。

### Q: ログインできません
- ニックネームとパスワードが正しいか確認してください
- 大文字・小文字を区別します
- 管理者にアカウントが作成されているか確認してください

### Q: 自分の対戦が表示されません
- ログインしているか確認してください
- 対戦がまだ作成されていない可能性があります（管理者がグループ設定を完了するまでお待ちください）

### Q: 言語を切り替えたい
ヘッダー右上の **EN/JA** ボタンで英語/日本語を切り替えられます。
