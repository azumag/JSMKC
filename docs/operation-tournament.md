# トーナメント全体 操作マニュアル

このマニュアルは、大会スタッフが JSMKC スコアシステムで大会を準備し、当日に進行し、終了後に結果を確認するための手順書です。画面の内部構造や技術的な説明ではなく、実際の運用順に沿って説明します。

- [MR 操作マニュアル](./operation-mr.md)
- [GP 操作マニュアル](./operation-gp.md)
- [TA 管理者マニュアル](./ta-admin-manual.md)
- [TA 参加者マニュアル](./ta-user-manual.md)

## 1. 最初に確認すること

### 1.1 役割

| 役割 | 主な作業 |
|------|----------|
| 管理者 | 大会作成、プレイヤー管理、予選設定、スコア修正、決勝開始、競技・総合ランキングの公開切り替え |
| プレイヤー | 参加者ページから自分のスコアやタイムを入力 |
| 観覧者 | 公開済みの大会ページで順位表や試合結果を閲覧 |

### 1.2 よく使うページ

| ページ | 用途 |
|--------|------|
| `/players` | プレイヤーの作成、編集、パスワード再発行 |
| `/tournaments` | 大会の作成、一覧確認 |
| `/tournaments/[id]` | 大会トップ、各競技への入口 |
| `/tournaments/[id]/participant` | プレイヤー向けの競技選択ページ |
| `/tournaments/[id]/ta` | タイムアタック |
| `/tournaments/[id]/bm` | バトルモード |
| `/tournaments/[id]/mr` | マッチレース |
| `/tournaments/[id]/gp` | グランプリ |
| `/tournaments/[id]/overall-ranking` | 総合ランキング |

`[id]` には大会ごとの ID またはカスタム URL が入ります。参加者には、主催者が共有した URL をそのまま案内してください。

## 2. 大会前日までの準備

### 2.1 管理者でログインする

1. 画面右上の「ログイン」を開きます。
2. 「管理者」タブを選びます。
3. 管理者アカウントでログインします。
4. ヘッダーにログイン状態が表示されることを確認します。

管理者用のボタンが表示されない場合は、ログインしているアカウントが管理者ではない可能性があります。

### 2.2 プレイヤーを登録する

1. トップまたはメニューから「プレイヤー」を開きます。
2. 「プレイヤー追加」を押します。
3. 名前、ニックネーム、国などを入力します。
4. 作成後に発行されるパスワードを、本人に安全な方法で伝えます。

パスワードを忘れたプレイヤーがいる場合は、プレイヤー管理画面から「パスワードリセット」を行い、新しいパスワードを本人へ伝えてください。

### 2.3 大会を作成する

1. 「トーナメント」を開きます。
2. 「新しいトーナメント」を押します。
3. 大会名と日付を入力します。
4. 必要に応じてカスタム URL を入力します。
5. 二重報告を使う場合は、両プレイヤー確認の設定を有効にします。
6. TA の自己編集可否を確認します。
7. 「作成」を押します。

作成直後の大会は、観覧者には表示されない未公開状態です。準備が終わるまでは公開しなくて構いません。

### 2.4 各競技の参加者を設定する

大会ページから TA、BM、MR、GP の各ページを開き、競技ごとに参加者を設定します。

| 競技 | 準備すること |
|------|--------------|
| TA | 「プレイヤー設定」で参加者、シード、ペアを設定 |
| BM | 「グループ設定」で参加者、シード、グループ数を設定 |
| MR | 「グループ設定」で参加者、シード、グループ数を設定 |
| GP | 「グループ設定」で参加者、シード、グループ数を設定 |

BM/MR/GP では、グループ設定を保存すると予選の総当たり試合が作成されます。設定をやり直すと既存の予選試合や入力済み結果に影響するため、当日運用中の再設定は慎重に行ってください。

### 2.5 プレイヤーに案内する

プレイヤーには、次の情報を事前に共有します。

- ログイン URL
- ニックネームと初期パスワード
- 大会の参加者ページ URL
- 自分が参加する競技
- スコア入力は本人のプレイヤーログインで行うこと
- 入力ミスに気づいた場合は管理者へ連絡すること

管理者でログインしている状態では、参加者向けスコア入力ページからプレイヤー本人としての入力はできません。管理者は各競技のメインページで入力・修正します。

## 3. 公開・未公開の考え方

### 3.1 公開状態とは

公開状態は、観覧者が各競技ページや総合ランキングを見られるかどうかを切り替えるためのものです。

| 状態 | 観覧者 | 管理者 |
|------|--------|--------|
| 未公開 | 大会一覧、競技ページ、総合ランキングから見えない | いつでも確認・編集できる |
| 公開 | 順位表や試合結果を見られる | 引き続き管理できる |

プレイヤー向けの入力ページは、公開ページとは別に運用されます。前日準備で参加者に入力ページを案内する場合は、ログイン方法も一緒に伝えてください。

### 3.2 公開・未公開を切り替える

各競技ページと総合ランキングページの上部に、管理者だけが使える公開スイッチがあります。スイッチ横の表示が「公開」なら観覧者に見える状態、「未公開」なら観覧者には見えない状態です。

1. 対象の競技ページ、または総合ランキングページを開きます。
2. 右上付近の公開スイッチをオンにします。
3. スイッチ横の表示が「公開」になり、タブ上の「未公開」表示が消えたことを確認します。

未公開に戻したい場合は、同じスイッチをオフにします。誤って公開した場合や、準備が間に合っていない場合に使います。

### 3.3 推奨する公開タイミング

| タイミング | 操作 |
|------------|------|
| 大会開始前 | 全競技を未公開のまま準備 |
| TA 開始時 | TA を公開 |
| BM 開始時 | BM を公開 |
| MR 開始時 | MR を公開 |
| GP 開始時 | GP を公開 |
| 結果確認後 | 必要に応じて総合ランキングを公開 |

公開は必ずしも全競技を一度に行う必要はありません。進行に合わせて、見せたい競技や総合ランキングだけを公開してください。

## 4. 当日の基本運用

### 4.1 開始前チェック

- [ ] 管理者でログインできる
- [ ] 全プレイヤーが作成済み
- [ ] 各競技の参加者設定が終わっている
- [ ] プレイヤーに参加者ページ URL を共有済み
- [ ] TA のペア設定とタイム入力方針を確認済み
- [ ] BM/MR/GP のグループと試合一覧を確認済み
- [ ] 公開したい競技と総合ランキングだけが公開状態になっている

### 4.2 競技中に見る場所

| 見たいもの | 見る場所 |
|------------|----------|
| 予選順位 | 各競技ページの「順位表」 |
| 試合一覧 | BM/MR/GP の「試合一覧」 |
| プレイヤーの報告状況 | 試合一覧の報告表示 |
| 同順位・同点の警告 | 順位表上部の警告表示 |
| 配信用 TV 番号 | 試合一覧または決勝ブラケットの TV# |
| 総合順位 | 総合ランキング |

### 4.3 スコア修正の基本

プレイヤー入力に誤りがあった場合、管理者が各競技ページから修正します。

1. 対象競技の「試合一覧」を開きます。
2. 対象試合の「スコア入力」または「結果入力」を開きます。
3. 正しい結果を入力して保存します。
4. 順位表が更新されたことを確認します。

二重報告が有効な大会では、両プレイヤーの報告が一致すると自動確定します。不一致の場合は、管理者が実際の結果を確認して保存します。

### 4.4 予選を確定する

BM/MR/GP では、予選が終わったら「予選を確定」を押します。

予選を確定すると、プレイヤーと管理者の予選スコア編集がロックされます。決勝ブラケットを作る前の最終確認として使ってください。

修正が必要になった場合は「予選のロック解除」を押してから修正し、修正後にもう一度「予選を確定」します。

### 4.5 同順位が出た場合

順位表に同順位の警告が出た場合は、サドンデスプレーオフなど大会ルールに沿って順位を決めます。

1. 警告が出ているグループを確認します。
2. 「プレーオフ結果を記録」を開きます。
3. プレイヤーを最終順位順に並べます。
4. 保存します。
5. 順位表の警告が消えたことを確認します。

---

## 5. 決勝へ進める

BM/MR/GP では、予選結果から決勝ブラケットを作成します。

1. 予選順位を確認します。
2. 同順位の警告が残っていないことを確認します。
3. 「予選を確定」を押します。
4. 参加人数に応じて次のボタンを押します。
   - 16名以下: 「決勝ブラケットを生成」
   - 17名以上: 「バラッジ開始 (Top 24)」
5. 「決勝へ」または「トーナメントを見る」から決勝ページを開きます。

ブラケットを作成したあとに予選結果を変える場合は、ブラケットリセットが必要になることがあります。リセットすると既存の決勝・バラッジ結果が削除されるため、必ず進行責任者に確認してから実行してください。

---

## 6. 配信向けの操作

試合一覧や決勝ブラケットには TV# を設定できる箇所があります。

- TV# は、どの配信台で行う試合かを示す番号です。
- 選択すると保存されます。
- 「配信に反映」を押すと、配信用表示に対象試合を反映します。
- TV3/TV4 は画面仕様上、反映対象外になる場合があります。

配信担当と運営担当で、TV# の使い方を事前に決めておくと当日の混乱を減らせます。

## 7. 結果確認とエクスポート

### 7.1 総合ランキングを確認する

1. 大会ページから「総合」を開きます。
2. 全4競技の合計ポイントを確認します。
3. 必要に応じて「再計算」を押します。

総合ポイントは、各競技の予選ポイントと決勝ポイントを合算して表示します。

### 7.2 データを保存する

大会ページ上部の「全データエクスポート」から、結果データをダウンロードできます。

エクスポートに失敗した場合は、次を確認してください。

- 管理者でログインしているか
- セッションが切れていないか
- 通信が安定しているか
- もう一度実行しても同じエラーになるか

## 8. よくあるトラブル

### 管理者用ボタンが見えない

管理者アカウントでログインしているか確認してください。プレイヤーとしてログインしている場合、管理者用の操作は表示されません。

### プレイヤーがログインできない

ニックネームとパスワードを確認し、必要であればプレイヤー管理からパスワードをリセットしてください。

### 参加者ページに試合が出ない

該当競技のグループ設定が終わっているか、本人がその競技に登録されているかを確認してください。すでに完了済みの試合しかない場合も、未完了欄には表示されません。

### スコア入力を間違えた

管理者が各競技ページから修正します。予選確定後はロック解除が必要です。

### 決勝ボタンが出ない

予選が未確定、参加者が不足している、または同順位の警告が残っている可能性があります。順位表と警告表示を確認してください。

### 公開したのに見えないと言われた

対象競技のページで「未公開」表示が消えているか確認してください。プレイヤーや観覧者が古いページを開いたままの場合は、ページを更新してもらってください。
