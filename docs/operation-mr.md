# マッチレース 操作マニュアル

このマニュアルは、マッチレース（MR）を大会スタッフが運用するための手順書です。MR は 1 対 1 のレース対戦で、予選では各試合 4 レースを行います。

---

## 1. マッチレースの流れ

1. 管理者が参加者を選び、グループを作成します。
2. 予選はグループ内の総当たりで行います。
3. 予選の各試合には、4 コースが事前に割り当てられます。
4. 各レースの勝者を入力すると、試合スコアが自動計算されます。
5. 予選終了後、同順位を解決して「予選を確定」します。
6. 決勝ブラケットまたは Top 24 バラッジを開始します。

### 1.1 使うページ

| ページ | 用途 |
|--------|------|
| `/tournaments/[id]/mr` | 予選順位、試合一覧、管理操作 |
| `/tournaments/[id]/mr/participant` | プレイヤー本人の結果報告 |
| `/tournaments/[id]/mr/match/[matchId]` | 試合詳細の共有ページ |
| `/tournaments/[id]/mr/finals` | 決勝ブラケット |

---

## 2. 予選準備

### 2.1 グループを作成する

1. MR ページを開きます。
2. 「グループ設定」を押します。
3. 参加させるプレイヤーにチェックを入れます。
4. 必要に応じてシード番号を入力します。
5. グループ数を選びます。
6. 「シード順で振分け」を押すと、シードをもとに自動配置できます。
7. 内容を確認して保存します。

保存すると、グループ内の総当たり試合が自動で作成されます。各試合には 4 コースが割り当てられます。

### 2.2 グループをやり直す

「グループ編集」から既存設定を変更できます。ただし、試合が再作成される場合があり、入力済みの結果に影響します。

当日中にやり直す場合は、すでに案内済みの対戦や配信台に影響がないか確認してください。

---

## 3. 予選結果入力

### 3.1 入力ルール

| 項目 | 内容 |
|------|------|
| レース数 | 4 レース固定 |
| 入力方法 | 各レースで勝者を選ぶ |
| 入力例 | `4-0`, `3-1`, `2-2`, `1-3`, `0-4` |
| 引き分け | `2-2` は有効 |
| クリア | 管理者は `0-0` で試合結果を消すことができます |
| BREAK | 相手が BREAK の場合は自動で `4-0` |

MR 予選は 4 レース固定です。古い案内で「5 レース」と書かれている資料がある場合は、このマニュアルの 4 レースを正としてください。

### 3.2 管理者が入力する

1. MR ページの「試合一覧」を開きます。
2. 対象試合の「結果入力」を押します。
3. 各レースのコースを確認します。
4. 各レースで勝者を選びます。
5. 自動計算されたスコアを確認します。
6. 「結果保存」を押します。
7. 順位表に反映されたことを確認します。

### 3.3 プレイヤーが報告する

プレイヤーは `/tournaments/[id]/mr/participant` から自分の未完了試合を確認し、結果を報告します。

二重報告が有効な大会では、両プレイヤーの報告が一致した時点で確定します。不一致の場合は、管理者が実際の結果を確認して修正します。

---

## 4. 試合一覧の見方

| 表示 | 意味 |
|------|------|
| `- - -` | まだ結果がありません |
| `3 - 1` | 確定済みの結果 |
| 報告欄 | プレイヤーからの報告状況 |
| TV# | 配信台番号 |
| BREAK | 不戦勝の相手 |

MR の試合詳細ページでは、割り当てられたコースや現在のスコアを確認できます。共有リンクとして使う場合も、スコア入力そのものは参加者ページまたは管理画面から行います。

---

## 5. 同順位が出た場合

順位表に同順位の警告が出たら、サドンデスプレーオフなど大会ルールに従って順位を決めます。

1. 警告が出ているグループを確認します。
2. 「プレーオフ結果を記録」を開きます。
3. プレイヤーを最終順位順に並べます。
4. 保存します。
5. 順位表の警告が消えたことを確認します。

保存した順位は、決勝進出者の並びに使われます。

---

## 6. 予選を確定する

全試合が終わり、順位が正しいことを確認したら「予選を確定」を押します。

予選確定後は、スコア編集がロックされます。修正が必要な場合は「予選のロック解除」を押し、修正後にもう一度「予選を確定」します。

---

## 7. 決勝を開始する

### 7.1 通常決勝

予選通過者が 16 名以下の場合は、「決勝ブラケットを生成」を押します。作成後は「トーナメントを見る」または「決勝へ」から決勝ページに移動します。

### 7.2 Top 24 バラッジ

予選通過者が 17 名以上の場合は、「バラッジ開始 (Top 24)」を押します。バラッジ結果を入力したあと、決勝ブラケットへ進みます。

### 7.3 決勝の試合形式

MR 決勝はダブルイリミネーション形式です。試合カードを開いてレース結果を入力すると、勝者・敗者が次の試合へ進みます。

| 段階 | 目安 |
|------|------|
| バラッジ | 先に 3 勝、後半は先に 4 勝 |
| 決勝序盤 | 先に 5 勝 |
| 中盤 | 先に 7 勝 |
| Winners Final / Losers Final / Grand Final | 先に 9 勝 |

画面上の入力欄に従って、必要な勝利数に達した結果を保存してください。

---

## 8. よくあるトラブル

### 4 レースを全部入れる必要がありますか

はい。MR 予選は 4 レース固定です。`2-2` の引き分けも正式な結果として保存できます。

### コースが思っていたものと違う

予選では試合ごとにコースが割り当てられます。運営上変更が必要な場合は、管理者が入力時に内容を確認してください。

### 予選確定後に修正したい

「予選のロック解除」を押してから修正します。修正後は再度「予選を確定」してください。

### 決勝ボタンが出ない

予選が未確定、参加者が不足している、同順位警告が残っている、またはブラケットがすでに存在する可能性があります。

- [ ] ブラケットを生成した

- [ ] 配信用 TV# を確認した
