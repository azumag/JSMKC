## 1. 概要

### 画面構成

| 画面 | パス | 用途 |
|------|------|------|
| MR予選 | `/tournaments/[id]/mr` | 予選グループステージ管理 |
| MR決勝 | `/tournaments/[id]/mr/finals` | ダブルイリミネーション決勝 |
| スコア入力 | `/tournaments/[id]/mr/participant` | プレイヤー自己報告 |
| 試合詳細 | `/tournaments/[id]/mr/match/[matchId]` | 個別試合の詳細 |

### スコアルール (§6)
- 1対1の5レース対戦
- 各レースでコースを選択し、1位と2位で勝敗決定
- 予選: コースはシステムがランダム割当（§10.5）
- 決勝: 主催者がアナウンス（§6.3）

## 2. 管理者フロー

### 2.1 グループ設定

1. MR予選ページ → **「グループ設定」** ボタンをクリック
2. プレイヤーを選択し、グループに割り当て
3. **「グループ＆試合作成」** をクリック
4. ラウンドロビン試合が自動生成（コースもランダム割当）

### 2.2 スコア入力

1. **「試合一覧」** タブで試合を選択
2. 各レースのコースと各プレイヤーの順位（1位/2位）を入力
3. 勝敗は自動計算（position1 < position2 → Player1の勝ち）
4. **「結果保存」** をクリック

### 2.3 コース割当

- §10.5: 予選では全20コースをシャッフルし、試合ごとに4コースずつ事前割当
- 割り当てられたコースは試合詳細で確認可能

## 3. プレイヤーフロー

### 3.1 レース結果報告

1. MR予選ページの **「Enter Score」** → participant ページ
2. 未完了の自分の試合が表示
3. **「Add Race」** ボタンでレース結果を追加
4. コース選択 + 各プレイヤーの順位（1位/2位）を入力
5. 最大5レースまで
6. **「Submit Match Result」** をクリック

### 3.2 Dual Report

BMと同じ二重報告システム。両プレイヤーのスコアが一致すれば自動確定。

---

## 4. 観覧者フロー

- MR予選ページで順位表・試合結果を閲覧（ログイン不要）
- グループ未設定時は待機メッセージ表示

---

## 5. 決勝ステージ

### 5.1 ブラケット生成

- MR決勝ページ → **「Generate Bracket」**
- Top 8 / Top 16 を選択可能
- ダブルイリミネーション方式

### 5.2 決勝レース入力

- 管理者がレースごとにコースと順位を入力
- §6.3: 決勝のコース・レース数は主催者がアナウンス

---

## 6. スコアルール

| 項目 | 予選 | 決勝 |
|------|------|------|
| レース数 | 5 | 主催者指定 |
| コース | ランダム割当 | 主催者指定 |
| 勝利条件 | 5レース中の勝ち数 | ブラケット |
| BYE | 4-0 自動完了 | なし |
| 順位計算 | 勝ち点 → 得失差 → H2H | ブラケット |
